Manuelle Installation

  1. Frische Installation von Gothic 2 DNDR, also ohne mods. Erhältlich bei Steam, etc
  2. Patch 2.6 drüber installieren: https://www.worldofgothic.de/dl/download_80.htm
  3. Patch 2.6 Fix drüber installieren: https://www.worldofgothic.de/dl/download_278.htm
  4. SystemPack 1.8 drüber installieren: https://www.worldofgothic.de/dl/download_597.htm
  5. .mod Dateien runterladen und in eurem Data-Ordner (Gothic II → Data) kopieren:
    • HoK_anims.mod
    • HoK_meshes.mod
    • HoK_scripts.mod
    • HoK_textures.mod
    • HoK_worlds.mod
    • HoK_sounds.mod
  6. GMP Launcher Downloaden und im neuen Ordner entpacken: GMP_Client_1.6.0
  7. Launcher starten und Verbindung eingeben
    • Alias: HoK
    • URL: 85.215.60.15
    • Port: 28960
  8. Im Launcher oben links unter Settings den Gothic II Pfad auswählen
  9. Doppelklick auf den Server
  10. Im Spiel könnt ihr mit /register <Accountname> <Passwort> einen Account erstellen
  11. Mit erstelltem Account könnt ihr mit /newchar <Charactername> einen Charakter erstellen.
  12. Der Server merkt sich euch via Cookie, es gibt also ein Autologin
