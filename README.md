# History of Khorinis Roleplay Website

Autor: **NoNameGames - Lou**

## Wichtige Dateien

- `index.html` – Startseite
- `aktuelles.html` – Übersicht aktueller Beiträge
- `aktuelles/` – einzelne Beiträge
- `data/aktuelles.json` – zentrale Inhaltsliste für Aktuelles
- `assets/css/styles.css` – aktive Anpassungen
- `assets/css/compatibility.css` – bestehende Layout-Kompatibilität
- `assets/images/` – aktive Bild- und UI-Assets

## Neue Aktuelles-Beiträge

1. Eintrag in `data/aktuelles.json` ergänzen.
2. Eine passende HTML-Datei unter `aktuelles/` anlegen.
3. Karte auf `index.html` und `aktuelles.html` ergänzen.

Die veralteten Änderungsprotokolle, Prüfberichte und nicht mehr verwendeten Button-Assets wurden entfernt.
